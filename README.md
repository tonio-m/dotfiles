# dotfiles

I use i3-gaps with i3blocks and rofi, and kitty as terminal emulator.
For editor I use neovim with the CoC plugin.

```
home
 └── user
     ├── .config
     │   ├── i3
     │   │   ├── config
     │   │   └── rofi.sh
     │   ├── i3blocks
     │   │   ├── batterybar
     │   │   ├── config
     │   │   ├── time
     │   │   └── wifi
     │   ├── kitty
     │   │   ├── kitty.conf
     │   │   └── theme.conf
     │   └── nvim
     │       ├── autoload
     │       │   └── plug.vim
     │       └── init.vim
     ├── .xinitrc
     └── .zshrc
```
